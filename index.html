Using JavaScript, the user is prompted to choose a style for the page.
Namely:

  • Background for the page;
  • Font style on the page;
  • Alignment for the h1 header;
  • Background for the paragraph with links;
  • Color of the text in the paragraph with links and for the links (the color of the text for the links is assigned in the loop);
  • Color, size, and thickness of the text in the main element;
  • Marker type for a list on the page.

Also, instead of links to the specific websites specified, the navigation displays links to three websites that the user enters.

For example, the user entered the address football24.ua in the prompt() window. In this case, a link is created that leads to the website football24, and the text of this link, which can be used to go to the specified website, is also football24.ua.

Thus, the user is prompted to enter ten page settings, after which they are applied.
