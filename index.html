Мої улюблені сайти:

  • W3Schools
  • Stack Overflow

За допомогою JavaScript користувачу запропонувано вибрати стиль для сторінки.
А саме:

  • фон для сторінки;
  • тип шрифта на сторінці;
  • вирівнювання для заголовка "h1";
  • фон для параграфа з посиланнями;
  • колір тексту у параграфі з посиланнями і для самих посилань, при цьому колір тексту для посилань призначено у циклі;
  • колір, розмір, товщину тексту у елементі "section";
  • тип маркера для маркованого списку на сторінці.

Окрім того, замість посилань на вказані конкретні сайти, у панелі навігації виведено посилання на три сайти, які вводить користувач.

Наприклад, користувач у віконце "prompt()" ввів адресу football24.ua. В такому разі створюється посилання, яке веде на сайт football24, а текстом цього посилання (за яким можна перейти на вказаний сайт) також є football24.ua.

Таким чином, користувачу запропонувано ввести десять налаштувань сторінки, після чого їх застосовано.
