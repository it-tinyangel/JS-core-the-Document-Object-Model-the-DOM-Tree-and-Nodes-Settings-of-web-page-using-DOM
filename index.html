Using JavaScript, the user is prompted to choose a style for the page. Namely:

  • background for the page;
  • font style on the page;
  • alignment for header "h1";
  • background for paragraph with links;
  • color of the text in the paragraph with links and for the links, while the color of the text for the links is assigned in the loop;
  • color, size, thickness of the text in the "div" element;
  • marker type for a list on a page.

In addition, instead of links to the specified specific sites, the navigation bar displays links to three sites entered by the user.

For example, the user entered the address football24.ua in the "prompt()" window. In this case, a link is created that leads to the football24 site, and the text of this link, which you can use to go to the specified site is also football24.ua.

Thus, the user is prompted to enter ten page settings, after which they are applied.
