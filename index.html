За допомогою JavaScript користувачу запропонувано вибрати стиль для сторінки.
А саме:

  • фон для сторінки;
  • тип шрифта на сторінці;
  • вирівнювання для заголовка h1;
  • фон елемента nav з посиланнями;
  • колір тексту в елементі nav з посиланнями і для самих посилань, при цьому колір тексту для посилань призначено у циклі;
  • колір, розмір, товщину тексту в елементі main;
  • тип маркера для списку на сторінці.

Окрім того, замість посилань на вказані конкретні веб-сайти, у панелі навігації виведено посилання на три веб-сайти, які вводить користувач.

Наприклад, користувач у віконце prompt() ввів адресу football24.ua. В такому разі створюється посилання, яке веде на веб-сайт football24, а текстом цього посилання, за яким можна перейти на вказаний веб-сайт, також є football24.ua.

Таким чином, користувачу запропонувано ввести десять налаштувань сторінки, після чого їх застосовано.
